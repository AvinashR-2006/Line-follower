# Line Follower

An ESP32 line follower robot project built around IR sensing and fast control loops.

![Line Follower Robot](assets/line-follower-robot.jpeg)

## Overview

This repository is a public showcase version of my line follower project. I am intentionally not publishing the complete competition-winning control logic, tuning values, and full motion strategy. Instead, this repo includes a safe starter firmware and system notes that show the project structure without exposing the full race bot implementation.

## What Is Included

- ESP32 PlatformIO project setup
- Basic IR sensor reading example
- Public hardware and tuning notes
- Project image and documentation

## What Is Not Included

- Competition-ready PID tuning
- Final turn strategy and recovery logic
- Winning bot firmware and optimized race behavior

## Getting Started

1. Install [PlatformIO](https://platformio.org/).
2. Open the repository in VS Code.
3. Build and upload the starter firmware.
4. Use the serial monitor to validate the IR sensor readings before writing your own control loop.

## Privacy Note

The complete race code is intentionally private. This helps protect original competition work while still sharing the project at a portfolio level.
